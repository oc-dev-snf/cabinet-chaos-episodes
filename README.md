# Cabinet Chaos Episodes

Weekly original political satire episode transcripts.

**Read online:** https://oc-dev-snf.github.io/cabinet-chaos-episodes/

## Format

- `episodes/YYYY-MM-DD-episode-XXX.md`
- Keep each episode self-contained and clearly titled.

## Contributing new episodes

1. Add a new markdown file under `episodes/`.
2. Use this template:
   - Title
   - Episode number
   - Cold open
   - Act One / Two / Three
   - Tag / End
3. Commit and push.

## Notes

- This is original parody writing inspired by chaotic political workplaces.
- No direct reproduction of copyrighted scripts.
